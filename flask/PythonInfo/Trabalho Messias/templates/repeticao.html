Estruturas de Repetição em Python

Compreendendo os laços e sua aplicação

O que são Estruturas de Repetição?

Em programação, estruturas de repetição (ou laços de repetição) permitem que blocos de código sejam executados múltiplas vezes de forma controlada, economizando linhas de código e automatizando tarefas repetitivas.

Tipos de Estruturas de Repetição em Python

  • while: repete enquanto uma condição for verdadeira.
  • for: percorre elementos de uma sequência (lista, string, range, etc).

Exemplo com while

contador = 1
while contador <= 5:
    print("Contador:", contador)
    contador += 1

Exemplo com for

for i in range(1, 6):
    print("Número:", i)

Quando usar?

Use while quando você não sabe exatamente quantas vezes a repetição vai ocorrer (condição indefinida), e for quando você já tem uma sequência ou número fixo de repetições.
