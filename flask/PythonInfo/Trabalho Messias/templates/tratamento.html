Tratamento de Exceções em Python

Como lidar com erros de forma segura e controlada

O que são Exceções?

Exceções são erros que ocorrem durante a execução de um programa. Em vez de encerrar o programa abruptamente, podemos tratá-las usando blocos de código especiais.

Estrutura básica do try-except

try:
    # código que pode causar erro
    numero = int(input("Digite um número: "))
    print("Você digitou:", numero)
except ValueError:
    print("Erro: valor inválido!")

Usando else e finally

try:
    resultado = 10 / 2
except ZeroDivisionError:
    print("Não é possível dividir por zero.")
else:
    print("Resultado:", resultado)
finally:
    print("Bloco finally executado.")

Capturando vários tipos de erro

try:
    arquivo = open("dados.txt", "r")
    conteudo = arquivo.read()
    arquivo.close()
except FileNotFoundError:
    print("Arquivo não encontrado.")
except IOError:
    print("Erro ao ler o arquivo.")

Por que tratar exceções?

  • Evita que o programa seja encerrado inesperadamente
  • Permite fornecer mensagens mais claras ao usuário
  • Melhora a robustez e confiabilidade do código
