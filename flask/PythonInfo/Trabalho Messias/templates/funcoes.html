Funções e Procedimentos em Python

Organizando o código com blocos reutilizáveis

O que são Funções?

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas ajudam a organizar e evitar repetição de código.

Como definir uma função em Python

def saudacao():
    print("Olá, seja bem-vindo!")

Para chamar a função:

saudacao()

Função com Parâmetros

def somar(a, b):
    return a + b

resultado = somar(5, 3)
print("Resultado:", resultado)

Procedimento vs Função

  • Procedimento: função que não retorna valor explicitamente (uso do print ou outras ações).
  • Função: retorna um valor com return.

Exemplo de Procedimento

def mostrar_mensagem(nome):
    print("Olá,", nome)

Por que usar funções?

  • Reutilização de código
  • Organização e clareza
  • Facilidade para testar e manter
