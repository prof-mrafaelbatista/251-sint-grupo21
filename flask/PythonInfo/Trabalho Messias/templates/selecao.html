Estruturas de Seleção

Conceitos e exemplos em programação

O que são Estruturas de Seleção?

Estruturas de seleção são mecanismos usados em programação para tomar decisões baseadas em condições. Elas permitem que o programa execute diferentes trechos de código dependendo do resultado de uma condição lógica.

Principais tipos de Estruturas de Seleção

  • If: Executa um bloco de código se a condição for verdadeira.
  • If...else: Executa um bloco se a condição for verdadeira e outro bloco se for falsa.
  • If...else if...else: Verifica múltiplas condições em sequência.
  • Switch: Seleciona a execução com base no valor de uma variável.

Exemplo em JavaScript

const idade = 18;

if (idade >= 18) {
  console.log("Você é maior de idade.");
} else {
  console.log("Você é menor de idade.");
}

Quando usar Estruturas de Seleção?

Use estruturas de seleção sempre que precisar executar diferentes ações dependendo de condições específicas, como validar dados, controlar fluxo do programa, reagir a entradas do usuário, entre outros.
